# Decoding

Counting Bytes L2R (0..5)
         Bits R2L (7...0)

## 6-Byte message

Byte 3, Bit 0 -> 1: hold, 0: free running


## 8-Byte message

Byte 0-1: Mode?
Byte 1-2: Range?
